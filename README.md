# 基于文档搜索

* 一些默认配置
  * 默认查询的块类型
    * 文档, 标题, 代码块, 数学公式块, 表格块, 段落块, html块
  * 默认排序方式
    * 类型 > 关联度（块内容与关键词的匹配次数） > 更新时间（最近更新的在前面）
* 当前不足
  * 不支持筛选类型
  * 不支持选择排序方式
  * 缺少异步幂等性
    * 当查询大数据时，接口还没响应。此时修改查询条件会发起新的接口立马响应，可能被前一次覆盖。
* 暂时无法解决的问题（没有思路）
  * 预览框无法高亮显示
  * 被隐藏快定位不准，应该在搜索结果标记
